# Redirect Maps

- Public map file: `legacy.map` (`$legacy_redirect_target`)
- Internal map file: `legacy.internal.map` (`$legacy_redirect_target_internal`)
- Gray mode selector: `gray-mode.conf` (`$legacy_redirect_mode` = `off|internal|all`)
- Generated by: `platform/scripts/generate-redirects.ts`

Nginx include paths:

- `/etc/nginx/conf.d/redirects/legacy.map`
- `/etc/nginx/conf.d/redirects/legacy.internal.map`
- `/etc/nginx/conf.d/redirects/gray-mode.conf`

`legacy.map` and `legacy.internal.map` are replaced atomically by the generator and validated by `validate-redirects.ts`.
